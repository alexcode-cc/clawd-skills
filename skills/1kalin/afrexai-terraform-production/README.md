# Terraform & Infrastructure as Code Production Engineering

Complete 14-phase Terraform production methodology for your AI agent. Zero dependencies.

## Install

```bash
clawhub install afrexai-terraform-production
```

## What's Inside

| Phase | Topic |
|-------|-------|
| 1 | Quick Health Check (8-signal /16 scoring) |
| 2 | Project Structure (recommended layout + 7 architecture rules) |
| 3 | State Management (remote backends, safety rules, key strategy) |
| 4 | Module Design (interface template, composition, 8 design rules) |
| 5 | Multi-Environment Strategy (comparison matrix, variable hierarchy, promotion) |
| 6 | Security Hardening (15-point checklist, secrets management, OIDC auth) |
| 7 | Testing Strategy (4-level pyramid, native TF tests, 7 testing rules) |
| 8 | CI/CD Pipeline (complete GitHub Actions, approval gates, cost estimation) |
| 9 | Resource Patterns (tagging, naming, dynamic blocks, cross-stack refs) |
| 10 | Drift Management (detection schedule, response playbook, ignore_changes guide) |
| 11 | Cost Optimization (Infracost CI, 7 cost strategies, tagging for allocation) |
| 12 | Advanced Patterns (Terragrunt, import blocks, multi-region, moved blocks) |
| 13 | Disaster Recovery (state recovery, migration checklist, breaking changes) |
| 14 | Quality Scoring (100-point rubric, 10 Commandments, 10 common mistakes) |

## Quick Start

Tell your agent:
> "Review my Terraform project for production readiness"

The agent runs the 8-signal health check, scores your project, and gives prioritized recommendations.
