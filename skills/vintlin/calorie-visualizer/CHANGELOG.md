# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-17
### Added
- Initial release of Calorie Visualizer.
- Multi-modal input support (Photo/Text).
- TDEE calculation via `USER.md` integration.
- Mermaid-based trend reporting.
- Local SQLite database for data persistence.
- "Polite Inquiry" logic for missing user profile data.
