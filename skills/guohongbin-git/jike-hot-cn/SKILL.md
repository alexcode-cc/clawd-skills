# 即刻热门话题监控

即刻热门动态监控，支持圈子分类、用户追踪。

## 功能

### 热门内容
- **热门动态** - 当前最火内容
- **精选推荐** - 编辑推荐
- **话题标签** - 热门话题追踪

### 圈子分类
- **科技圈** - 科技数码讨论
- **投资圈** - 理财投资心得
- **生活圈** - 日常分享
- **职场圈** - 工作相关

## 使用方式

### 获取热门动态

```
获取即刻今日热门
```

### 按圈子筛选

```
获取即刻科技圈热门
```

## 输出格式

```
💛 即刻热门动态

1. 今天换了个新机械键盘，手感太棒了！
   📂 数码圈 | 💛 1.2k | 💬 89

2. 分享一下我的投资心得
   📂 投资圈 | 💛 987 | 💬 156
```
